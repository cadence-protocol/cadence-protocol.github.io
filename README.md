# cadence-protocol.github.io

Teaser website for [Cadence Protocol](https://cadenceprotocol.build) — the open ERC standard and automation layer for onchain recurring payments.

Served via GitHub Pages at [cadenceprotocol.build](https://cadenceprotocol.build).

## Structure

```
index.html      # Single-page teaser site
logo/           # Logo assets
CNAME           # Custom domain configuration
```

## Local preview

Open `index.html` directly in a browser — no build step required.
